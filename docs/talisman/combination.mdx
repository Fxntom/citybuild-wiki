# Kombination von Talismanen

Über das unterste Item im Talisman Menü gelangt man in das hier abgebildete GUI. Dort kann man auf die zwei Barrieren je einen Talisman der gleichen Seltenheit legen und über das Plus kombinieren. Gelingt dies, so erhält man einen Talisman mit einer höheren Stufe, scheitert es, erhält man einen Talisman zurück.
Kombiniert man zwei Legendäre Talismane erhält man einen zufälligen anderen Legendären.
