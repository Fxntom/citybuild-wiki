# Übersicht

Ein **Talisman** verleiht spezielle Effekte. Diese gibt es in **vier Seltenheitsstufen:** 
- Gewöhnlich
- Selten
- Episch
- Legendär

Umso höher diese Stufe, desto besser ist der Effekt des jeweiligen Talismans. 

import Admonition from '@theme/Admonition';

<Admonition type="info" title="Hinweis!">
Beachte, dass es nicht jeden Talisman in jeder Seltenheitsstufe gibt! Eine genaue Liste findest du Ingame im Menü.
</Admonition>

Insgesamt existieren aktuell **27** verschiedene Talismane, die meisten davon in mehreren Seltenheiten. Der Befehl ```/talisman menu``` zeigt dir alle Talismane und ihre Effekte in Abhängigkeit ihrer Seltenheit an. Dieser öffnet folgendes GUI:
