# Plot Config
Über "/plot config" kann man auf seiem Plot ein Menü aufrufen, in welchem man eine Vielfalt von Einstellungen für das Plot treffen kann, manche Einstellungen lassen dabei nur mithilfe eines Ranges einstellen. 
So lässt sich z.B. einstellen, dass jede auf Werkbänke oder Zaubertische zugreifen kann oder sogar das niemand Items aufsammeln kann.
