# Showcases
In einem Showcase haben andere Spieler die Möglichkeit, den Inhalt bestimmter Kisten oder Shulkerboxen anzusehen, ohne dass sie die Items entnehmen können oder Berechtigungen für den Plot haben.  
Der Inhalt von Shulkerboxen in einem Showcase kann durch die Tastenkombination **Shift + Rechtsklick** eingesehen werden.

**Showcase erstellen**\
Um einen Showcase zu erstellen, platziere ein Schild an einer Kiste oder Shulkerbox und schreibe in einer beliebigen Zeile „[Showcase]“.
