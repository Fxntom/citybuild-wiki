# Befehle

## Befehle für alle CityBuild Spieler

- **/plot tp [Alias/QuickID]**: Teleportiert dich zu einem deiner Grundstücke mit der angegebenen Quick-ID bzw. dem entsprechenden Alias.
- **/plot tp Spieler [Alias/QuickID]**: Teleportiert dich zu einem bestimmten Grundstück des angegebenen Spielers.
- **/plot list**: Zeigt eine Liste deiner Grundstücke an.
- **/plot listall**: Zeigt eine Liste aller Grundstücke an, auf denen du Mitglied bist.
- **/plot roles**: Listet alle verfügbaren Grundstückrollen auf.

## Befehle für Plotwelten

* Diese Befehle erfordern möglicherweise ein Vorteilspaket, Rang oder Rechte aus der Crate.

- **/plot auto**: Kauft ein zufälliges Grundstück.
- **/plot claim**: Kauft das Grundstück, auf dem du dich befindest.
- **/plot info**: Zeigt Informationen über das Grundstück an, auf dem du dich befindest.
- **/plot setrole Spieler Rolle**: Weist einem Spieler eine Rolle auf deinem Grundstück zu.
- **/plot removerole Spieler**: Entfernt einem Spieler eine Rolle auf deinem Grundstück.
- **/plot setalias Alias**: Ändert den Alias deines Grundstücks.
- **/plot removealias**: Entfernt den Alias deines Grundstücks.
- **/plot setprivacy private/public**: Ändert die Privacy-Einstellung deines Grundstücks.
- **/plot lock**: Setzt die Privacy-Einstellung deines Grundstücks auf privat. Nur Mitglieder können es betreten.
- **/plot offer Spieler Geld**: Unterbreitet dem Besitzer eines Grundstücks ein Kaufangebot.
- **/plot offer toggle**: De-/aktiviert Kaufanfragen für das jeweilige Plot.
- **/plot merge**: Verbindet dein aktuelles Grundstück mit dem Grundstück in Blickrichtung.
- **/plot mergelimits**: Zeigt an, wie groß Plot-Merges in Abhängigkeit von Plot-Reihe auf der aktuellen Plotwelt sein können.
- **/plot delete**: Löscht dein Grundstück (Besitz abgeben und zurücksetzen).
- **/plot clear**: Setzt dein Grundstück zurück.
- **/plot kick Spieler**: Wirft einen Spieler von deinem Grundstück.
- **/plot ban Spieler**: Bannt einen Spieler von deinem Grundstück.
- **/plot unban Spieler**: Entbannt einen Spieler von deinem Grundstück.
- **/plot bans**: Zeigt alle vom Grundstück gebannten Spieler an.
- **/plot setspawn**: Setzt den Spawnpunkt deines Grundstücks.
- **/plot sethome**: Markiert ein Grundstück als Home-Grundstück.
- **/plot fly**: De-/aktiviert, ob alle Spieler auf diesem Grundstück fliegen können. Dafür wird das Fly-Perk aus einem Vorteilspaket benötigt¹.
- **/plot edge**: Ändert die Umrandung deines Grundstücks¹.
- **/plot wall**: Ändert die Wände deines Grundstücks¹.
- **/plot floor**: Ändert den Boden (Höhe -64) deines Grundstücks¹.
- **/plot hologram create/remove**: Erstellt/Entfernt ein Hologramm¹.
- **/plot title**: De-/aktiviert die Willkommensnachricht.
- **/plot center**: Teleportiert dich in die Mitte deines Grundstücks.
- **/plot border**: Umrandet das Grundstück, auf dem du stehst¹.
- **/plot deleteinfo**: Zeigt an, wann das Grundstück, auf dem man sich befindet oder auf das man schaut, gelöscht wird.
- **/plot move**: Zeigt Informationen zu einer laufenden Verschiebung eines Grundstücks an.
- **/plot move start**: Beginnt die Verschiebung eines Grundstücks.
- **/plot move confirm**: Bestätigt den Zielort während eines Verschiebeprozesses.
- **/plot move cancel**: Bricht den Verschiebevorgang ab.
- **/plot level**: Zeigt das Level des aktuellen Grundstücks an.
- **/plot togglenotify**: De-/aktiviert, ob du benachrichtigt werden willst, wenn jemand das aktuelle Grundstück betritt.
- **/plot limits**: Zeigt an, wie viele Plots und Merges du noch besitzen darfst. Abkürzung: /limits.
- **/plot entitylimits**: Zeigt an, wie viele Mobs und Objekte du noch auf deinem Plot besitzen darfst.
- **/plot buildall**: De-/aktiviert, dass jeder Spieler auf deinem Plot bauen und auf Kisten zugreifen kann.
- **/plot config**: Öffnet ein Menü, in dem verschiedenste Einstellungen für das Plot getätigt werden können.
- **/plot Biome set Biom**: Setzt das Biom des gesamten Plots auf das ausgewählte Biom¹.
- **/plot Biome setsupplot Biom**: Setzt das gesamte Subplot auf das ausgewählte Biom¹.
- **/plot Biome setxyz x¹ y¹ z¹ x² y² z² Biom**: Setzt an den bestimmten Koordinaten ein Biom - Aktuelle Koordinaten können mit Tab vervollständigt werden¹.
- **/plot Biome reset**: Setzt das Biom des gesamten Plots auf Plain (Standard) zurück.
- **/plot Biome resetsubplot**: Setzt das Biom des gesamten Plots auf Plain (Standard) zurück.
- **/plot setweather clear/rain**: Setzt das Wetter auf deinem Plot¹.
- **/plot bossbar create Nachricht**: Erstellt eine Bossleiste für dein Grundstück, die jeder auf deinem Grundstück sieht¹.
- **/plot bossbar remove**: Löscht die Bossleiste für dein Grundstück.
- **/plot logbuch**: Öffnet ein Inventar, in welchem die letzten 9 Besucher deines Grundstücks angezeigt werden.
- **/plot online**: Zeigt eine Liste von Spielern, die sich derzeit auf dem Plot befinden.
