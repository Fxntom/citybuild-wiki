# Hologramme
Hologramme ermöglichen es, visuelle Texte in der Luft zu platzieren.

Das Erstellen und Löschen von Hologrammen setzt die **Plot-Rolle Owner** oder **Co-Owner** sowie den **Premium-Rang** voraus. Um ein Hologramm zu erstellen, benutze den Befehl **/hologram create Text**, wobei du den gewünschten Text an dem Standort deiner „Skin-Füße“ platzierst. Mit verschiedenen **Farbcodes** kannst du den Text formatieren und gestalten.  
Zum Entfernen eines Hologramms kannst du **/hologram remove** verwenden. Dadurch wird das Hologramm entfernt, das sich an deinen „Füßen“ am nächsten befindet.

**Hologrammlimits**\
Du kannst pro Teilplot bis zu **drei Hologramme** erstellen. Dieses Limit kann unendlich oft durch das **Mergen** von Plots erweitert werden, wobei für jedes zusätzliche Merge **+3 Hologramme** pro Plot hinzugefügt werden. Zudem erhöht sich das Limit um **+1 weiteres Hologramm** pro Wegkreuzung, die zwischen vier Plots liegt und ins Merge einfließt.
