# Schatzkisten
Befindet man sich in einer Gilde können beim abbauen von Blöcken in der Farmwelt CityBuild-Schatzkisten gefunden werden, je höher dabei das Level deiner Gilde ist, desto wahrscheinlicher ist es, dass du Schatzkisten findest, die Wahrscheinlichkeit kann dabei durch einen bestimmten Talisman weiter erhöht werden. 
In Schatzkisten können eine Vielzahl an Items auftauschen, darunter auch Schmiedevorlagen (Armortrims) für Rüstung
