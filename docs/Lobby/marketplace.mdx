# Marktplatz
Am Marktplatz können eine Vielzahl an Items gegen kleines Geld verkauft werden, um sich so etwas Geld nebenbei zu verdienen. Die dabei tägliche Menge an möglichen Geld richtet sich dabei nach dem CityBuild-Level.
