# Gilden - Übersicht


Mit Gilden hast du die Möglichkeit, eine eigene Gemeinschaft zu gründen und gemeinsam im Level aufzusteigen, um verschiedene spannende Vorteile freizuschalten.

Nutze den Befehl /guild create [Name], um deine Gilde zu gründen. Du erhältst automatisch den Gründer-Status. 
Der Gildenname ist frei wählbar und kann nachträglich mit /guild rename [Name] geändert werden.

Spieler lassen sich mit /guild invite [Spieler] einladen. Eingeladene Spieler können Einladungen mit /guild accept [Gilde] annehmen oder mit /guild deny [Gilde] ablehnen. Um eine Gilde zu verlassen, 
verwende den Befehl /guild leave, es sei denn, du bist der Gründer. 
In diesem Fall muss die Gilde entweder mit /guild delete gelöscht oder der Gründer-Status mit /guild transfer [Spieler] an einen anderen Spieler übertragen werden.

Eine Gilde kann anfangs maximal drei Mitglieder haben. Wenn Gildenmitglieder den Premium-Rang besitzen, erhöht sich die Kapazität der Gilde um jeweils einen zusätzlichen Platz pro Premium-Mitglied. 
Mitglieder mit Supremium-Rang oder Teammitglieder des GommeHD.net-Netzwerks erweitern die Gilde um jeweils drei zusätzliche Plätze. 
Weitere Mitgliederplätze können zudem über die Gildenvorteile freigeschaltet werden.
