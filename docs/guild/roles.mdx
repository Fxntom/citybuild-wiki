# Rollen und deren Berechtigungen

Gildenmitglieder können mit dem Befehl /guild promote [Spieler] auf die nächsthöhere Rolle befördert und mit /guild demote [Spieler] auf die nächstniedrigere Rolle herabgestuft werden.\
\
Folgende Rollen gibt es und folgende Rechte haben diese:

| Berechtigungen           | Gründer         | Leiter           | Moderator        | Mitglied         | Gast            |
|--------------------------|-----------------|------------------|------------------|------------------|-----------------|
| Profit aller Gildenvorteile | ✓              | ✓               | ✓               | ✓               | ✓              |
| Zugriff auf den Gildenchat | ✓              | ✓               | ✓               | ✓               | ✓               |
| Zugriff auf das Backpack  | ✓              | ✓               | ✓               | ✓               | ✖              |
| Spieler einladen          | ✓              | ✓               | ✓               | ✖               | ✖              |
| Spieler befördern         | bis Leiter      | bis Moderator    | bis Mitglied     | ✖               | ✖              |
| Spieler degradieren       | ab Leiter       | ab Moderator     | ab Mitglied      | ✖               | ✖              |
| Spieler kicken            | bis Leiter      | bis Moderator    | bis Mitglied     | ✖               | ✖              |
| Gilde umbenennen          | ✓             | ✓               | ✖               | ✖               | ✖              |
| Banner ändern             | ✓              | ✓               | ✖               | ✖               | ✖              |
| MOTD ändern               | ✓              | ✓               | ✖               | ✖               | ✖              |
| Gilde übertragen          | ✓              | ✖               | ✖               | ✖               | ✖              |
| Gilde löschen             | ✓              | ✖               | ✖               | ✖               | ✖              |
